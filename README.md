**Airbnb Paris Listing Analysis**

**Project Overview :**
This project analyzes Airbnb listing data for Paris to uncover pricing patterns, neighborhood trends, host growth behavior, and long-term price evolution.
The objective is to extract meaningful business insights using data cleaning, exploratory data analysis (EDA), and visualization techniques in Python.

**Business Questions Addressed :**
Which Paris neighborhoods have the highest average listing prices?
How does accommodation capacity affect pricing?
How has Airbnb host registration changed over time?
What are the long-term price trends?

**Tools & Technologies :**
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook

**Key Insights :**
Central neighborhoods show significantly higher average prices.
Listings with higher accommodation capacity command premium pricing.
Host registrations increased sharply after 2015.
Listing prices have shown consistent long-term growth trends.

**Business Recommendations :**
Investors should focus on high-demand central neighborhoods.
Hosts can strategically price based on accommodation size.
Emerging neighborhoods present growth opportunities.
